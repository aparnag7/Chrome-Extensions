# Site-Blocker
A chrome extension that blocks the following websites:
<li> Netflix
<li> Amazon Prime
<li> Hotstar
<li> Facebook
<li> Instagram
<li> Twitter

## How to use? 
  <p> Clone the repository. Go to <a href "chrome://extensions/">chrome extensions</a>. Turn on developer's mode. Click 'Load Unpacked' and add <b>Site-Blocker</b>
  </p>
